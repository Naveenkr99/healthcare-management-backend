# healthcare-management

healthcare-management-backend is a RESTful backend application designed to support healthcare management systems.
It provides APIs for managing patients, doctors, appointments, and medical records, enabling secure and efficient handling of healthcare data.
The system follows a layered architecture with proper validation, error handling, and modular design to ensure scalability and maintainability.
Built a microservices-based service using Java, Spring Boot, postgreSQL and Apache Kafka , with REST API design.
Configured load balancers and API gateways for traffic routing and secured REST APIs using JWT-basedauthentication
